# A general-purpose digital data acquisition system(GDDAQ)

![Yi logo](Yilogo100.png)

本程序支持 CAEN 第二代数字化采集卡，包括 2745/2740/2730。支持不同模块 PHA/PSD/ZLE/DAW/SCOPE 固件混合使用。

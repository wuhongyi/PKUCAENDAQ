关于

本说明书仅适用于 CAEN 的第二代数字化采集卡 2745/2740/2730 系列采集卡。支持不同模块 PHA/PSD/ZLE/DAW/SCOPE/OPEN 固件混合使用。通用逻辑的实现需要 2495 可编程逻辑模块。

第二代数字化采集卡型号为 x27xx
